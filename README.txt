Cardio Can Wait — static site (GitHub Pages)

This version uses RELATIVE paths, so it works whether you publish as:
- https://<user>.github.io/  (user/organization site), or
- https://<user>.github.io/<repo>/ (project site)

Publish
1) Create a GitHub repo
2) Upload all files to the repo root
3) Settings → Pages → Deploy from branch → main / (root)

Logo
- Current logo is included as assets/logo.png
- Replace it anytime with your own file (same name) to update the site and favicon.
