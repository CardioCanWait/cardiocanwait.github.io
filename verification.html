Ownership verification

Cardio Can Wait

Some platforms ask you to prove you control the site by: (1) uploading a file at the website root, or (2) adding a specific <meta> tag to your homepage.

Option A: add a meta tag

If the platform gives you a meta tag, paste it into index.html inside the <head>.

<!-- Example: <meta name="google-site-verification" content="..."/> -->

Option B: upload a verification file

If they give you a file like google1234567890abcdef.html, place it next to index.html (website root) and redeploy.

Option C: .well-known

If they require something under /.well-known/, put it in the folder .well-known/ in this project.
